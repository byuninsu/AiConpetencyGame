# 길 만들기 퍼즐

잡다/마이더스 예시 화면을 참고해 만든 5x5 길 만들기 연습용 웹 퍼즐입니다.

## 실행

브라우저에서 `index.html`을 열면 바로 실행됩니다.

정적 서버로 띄우고 싶다면:

```powershell
python -m http.server 8080
```

그다음 `http://localhost:8080`으로 접속하면 됩니다.

## 현재 구성

- 5x5 내부 모든 칸을 직접 눌러 `/` 또는 `\` 울타리를 배치합니다.
- 같은 방향을 다시 누르면 해당 칸의 울타리가 지워집니다.
- 문제마다 `클릭 가능 횟수`와 `정답의 울타리 수`가 정해집니다.
- `제출` 버튼을 눌렀을 때만 정답 여부를 판정합니다.
- 정답은 `같은 색 연결 + 최소 울타리 수 일치`를 동시에 만족해야 합니다.
- 고정 3단계가 아니라, 최소 조건이 검증된 랜덤 문제 풀에서 계속 다음 문제를 생성합니다.
